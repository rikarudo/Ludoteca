Class: ImagemFilme

ImagemFilme(x, y, filme)

A classe ImagemFilme é uma subclasse de Grafico, servindo para representar filmes (videos) de forma embutida no canvas. Num nível básico, uma imagem de filme é definida por um ponto, correspondente, regra geral, ao seu canto superior esquerdo e pela própria imagem — isto é, o filme.

desenha(tela)

Este método desenha um video no canvas, usando como referência de posicionamento o ponto correspondente ao canto superior esquerdo desse vídeo. Note-se que o vídeo só é efetivamente reproduzido no canvas se o elemento video associado por via da propriedade filme estiver também a ser reproduzido.

reposicionaContra(outro, afastamento) → {boolean}

Este método testa a colisão entre este objecto gráfico e outro objecto gráfico e, quando tal se verifica, reposiciona este objecto gráfico, deixando-o encostado (ou próximo disso, consoante o afastamento) ao outro, evitando a sobreposição de ambos. O reposicionamento depende da posição imediatamente anterior deste objecto gráfico em relação ao outro. Neste momento, ainda só funciona devidamente com movimentos definidos pelo deltaX, pelo deltaY e pela gravidade.
Parameters:
Name Type Default Description
