desenhaGraficos(graficos, tela)

Esta função desenha um gráfico ou percorre um array de gráficos (instâncias ou objectos de subclasses da classe Grafico) e desenha-os no canvas especificado. A função funciona de forma recursiva, pelo que o array pode conter, ele próprio, também arrays em qualquer das suas posições.
Parameters:
Name Type Description
graficos Grafico Um array de gráficos ou mesmo apenas um único gráfico
tela Tela O canvas onde vão ser desenhados os gráficos
