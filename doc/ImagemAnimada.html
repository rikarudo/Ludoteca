Class: ImagemAnimada

ImagemAnimada(x, y, imagem, fotogramas, iteracoes, tiras)

A classe ImagemAnimada é uma subclasse de Grafico, servindo para representar imagens animadas (ou sprites). Num nível básico, uma imagem é definida por um ponto, correspondente, regra geral, ao seu canto superior esquerdo e pela própria imagem. Os sprites utilizados podem ter uma sequência horizontal (tira) ou várias, desde que os fotogramas sejam de dimensões idênticas.

Este método desenha uma imagem animada no canvas, usando como referência de posicionamento o ponto correspondente ao canto superior esquerdo dessa imagem animada, no caso de uma única tira. Havendo mais que uma tira, usará o ponto correspondente ao canto superior esquerdo dessa tira.
