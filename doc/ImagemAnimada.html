Class: ImagemAnimada

ImagemAnimada(x, y, imagem, fotogramas, iteracoes, tiras)

A classe ImagemAnimada é uma subclasse de Grafico, servindo para representar imagens animadas (ou sprites). Num nível básico, uma imagem animada é definida por um ponto, correspondente, regra geral, ao seu canto superior esquerdo, com os vários fotogramas a serem desenhados nessa posição, assumindo-se dimensões idênticas para cada um deles. Assim, os sprites utilizados podem ter uma sequência horizontal (tira) ou várias, desde que os fotogramas tenham as mesmas dimensões.

Methods

colide(outro, reposicionamento, afastamento) → {boolean}

Este método verifica se existe uma colisão entre este gráfico e outro. Tal acontece quando um dos pontos de um gráfico se encontra contido na área de outro. Para o efeito, são usados os pontos que definem o envelope de cada gráfico. Este método também permite, caso se verifique a colisão, reposicionar este gráfico, deixando-o encostado (ou próximo disso, consoante o afastamento) ao outro, evitando a sobreposição de ambos. O reposicionamento depende da posição imediatamente anterior deste gráfico em relação ao outro. [Neste momento, o reposicionamento ainda só funciona devidamente com movimentos definidos pelo deltaX, pelo deltaY e pela gravidade. Note-se também que as colisões são testadas em toda a área do gráfico, e não apenas em áreas específicas, como acontece no caso das caixas de colisão (hitboxes), que ainda não se encontra implementadas.]
