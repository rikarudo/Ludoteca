Class: Rectangulo

Rectangulo(x, y, largura, altura, preenchimentoopt, contornoopt, espessuraopt)

A classe Rectangulo é uma subclasse de Grafico, servindo para representar rectângulos. Num nível básico, um rectângulo é definido por um ponto, correspondente, regra geral, ao seu canto superior esquerdo e por uma largura e por uma altura.

colide(outro, reposicionamentoopt, afastamentoopt, saltosopt) → {boolean}

Este método verifica se existe uma colisão entre este gráfico e outro. Tal acontece quando um dos pontos de um gráfico se encontra contido na área de outro. Para o efeito, são usados os pontos que definem o envelope de cada gráfico. Este método também permite, caso se verifique a colisão, reposicionar este gráfico, deixando-o encostado (ou próximo disso, consoante o valor do afastamento) ao outro, evitando a sobreposição de ambos. O reposicionamento depende da posição imediatamente anterior deste gráfico em relação ao outro. [Neste momento, o reposicionamento ainda só funciona devidamente com movimentos definidos pelo deltaX, pelo deltaY e pela gravidade.]
Parameters:
Name Type Attributes Default Description
outro Grafico Outro gráfico para verificar se existe alguma colisão entre esse e este
reposicionamento boolean <optional>
 true Indicação de que este gráfico deve ser reposicionado (ou não) quando colide com o outro
afastamento number <optional>
 0 Afastamento a aplicar entre os dois gráficos no reposicionamento; idealmente, este deve ser compreendido entre zero (0) e um valor inferior aos valores de quaisquer deslocamentos horizontais e verticais
saltos number <optional>
 0 Número de saltos (omissões de iterações ou frames) em chamadas consecutivas a este método — pode ser útil definir um número de saltos superior a zero (0) em situações com verificações de colisões entre muitos gráficos em simultâneo, de forma a diminuir a exigência computacional; se reposicionamento=true, este parâmetro é ignorado, não havendo saltos, de modo a que o reposicionamento seja correctamente efectuado

Este método inverte o sentido da força da gravidade aplicada sobre o gráfico, contanto esta seja diferente de zero, pondo o valor interno de anguloGravidade, cujo co-seno é usado, em 180.
