Class: Media

(abstract) Media(media)

A classe Media é, na prática, uma classe abstracta, servindo apenas de base para as subclasses de Media. Para além disso, funcionará essencialmente como um envelope (wrapper) para elementos de audio e de video em eventuais subclasses.
