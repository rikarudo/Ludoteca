Class: Som

Som(som)

A classe Som funciona essencialmente como um invólucro para elementos HTMLAudioElement, facilitando a sua referenciação e também o uso dos métodos mais comuns para, por exemplo, o controlo da sua reprodução.

Constructor

new Som(som)

Construtor para criação de novos objectos do tipo Som. Inicializa a instância com o elemento HTML do tipo audio a ser representado.

Properties:
Name Type Description
som HTMLAudioElement O elemento do tipo HTMLAudioElement a ser representado.
posicao number O tempo actual de reprodução (em segundos) do som. Pode variar entre 0.0 e duracao. Se o valor atribuído estiver fora deste intervalo, será corrigido para 0.0 (se inferior) ou duracao (se superior).
duracao number A duração total do som (em segundos).
volume number O volume actual do som. Pode variar entre 0.0 e 1.0. Se o valor atribuído estiver fora deste intervalo, será corrigido para 0.0 (se inferior) ou 1.0 (se superior).

reproduz(inicioopt)

Inicia ou retoma a reprodução do elemento multimédia. Caso o parâmetro inicio seja true, a reprodução é iniciada desde o início; caso contrário, é retomada a partir do momento em que foi interrompida anteriormente (ou do momento indicado pelo tempo de reprodução), ou continua a sua reprodução (se já estava a ser reproduzido).
Parameters:
Name Type Attributes Default Description
inicio boolean <optional>
 false Indicação de que o elemento deve ser reproduzido do início; caso contrário, é reproduzido a partir do momento em que foi interrompida a sua reprodução anterior (ou do momento indicado pelo tempo de reprodução), ou continua a sua reprodução (se já estava a ser reproduzido).
