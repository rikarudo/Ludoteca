Class: Circulo

Circulo(x, y, raio, preenchimento, contorno, espessura)

A classe Circulo é uma subclasse de Grafico, servindo para representar círculos. Num nível básico, um círculo é definido por um ponto central e por um raio. Contudo, para que exista uniformidade no posicionamento das formas representadas pelas várias subclasses de Grafico, os círculos são também posicionados usando o canto superior esquerdo (de um quadrado imaginário que o contenha), em vez do seu centro — daí, o raio é somado ao x e ao y para fazer as devidas compensações de posicionamento para determinar o centro.

Methods

colide(outro) → {boolean}

Este método encontra-se implementado na superclasse Grafico, assumindo que todas as formas definidas nas suas subclasses são limitadas, de um modo genérico, por um rectângulo imaginário que engloba a totalidade do gráfico. Se este método, tal como se encontra definido, não for o mais apropriado para uma dada subclasse, pode ser sempre redefinido (overrided) nessa mesma subclasse.
