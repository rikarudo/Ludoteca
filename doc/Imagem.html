Class: Imagem

Imagem(x, y, imagem)

A classe Imagem é uma subclasse de Grafico, servindo para representar imagens. Num nível básico, uma imagem é definida por um ponto, correspondente, regra geral, ao seu canto superior esquerdo e pela própria imagem.

Methods

colide(outro, reposicionamento, afastamento) → {boolean}

Este método verifica se existe uma colisão entre este gráfico e outro. Tal acontece quando um dos pontos de um gráfico se encontra contido na área de outro. Para o efeito, são usados os pontos que definem o envelope de cada gráfico. Este método também permite, caso se verifique a colisão, reposicionar este gráfico, deixando-o encostado (ou próximo disso, consoante o afastamento) ao outro, evitando a sobreposição de ambos. O reposicionamento depende da posição imediatamente anterior deste gráfico em relação ao outro. [Neste momento, o reposicionamento ainda só funciona devidamente com movimentos definidos pelo deltaX, pelo deltaY e pela gravidade. Note-se também que as colisões são testadas em toda a área do gráfico, e não apenas em áreas específicas, como acontece no caso das caixas de colisão (hitboxes), que ainda não se encontra implementadas.]
Parameters:
Name Type Default Description
outro Grafico Outro gráfico para verificar se existe alguma colisão entre esse e este
reposicionamento boolean true Indicação de que este gráfico deve ser reposicionado (ou não) quando colide com o outro
afastamento number 0 Afastamento a aplicar entre os dois gráficos no reposicionamento; idealmente, este deve ser compreendido entre zero (0) e um valor inferior aos valores de quaisquer deslocamentos horizontais e verticais

contemPonto(x, y) → {boolean}

Este método verifica se um dado ponto está contido na área do envelope gráfico, assumindo-se que o gráfico é um polígono. Assim, tal acontece quando um segmento de recta com início no ponto especificado intersecta um número ímpar de arestas do polígono.
